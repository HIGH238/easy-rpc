Easy RPC


Here are a few things to get you started:

• Client ID: This is the ID of your application. It can be obtained from the Discord Developer Portal
                   Note: The name of this application directly affects the name of the RP

• Start & Stop Times: These will be the start and stop times of the RP. Just write the start time to get 'elapsed' time and write both to get 'remaining' time
                                     Note: These time stamps are in Unix Epoch and are in Milliseconds

• Image Keys: These should be filled in with the asset keys that you set in Discord Developer Portal

• Image Texts: These texts will appear when you hover over the large and small images respectively

• Party ID: This will be the ID of the party. It can be left empty
                  Note: You can fill this in with a random string if you want to show the 'Ask to join' button

• Party Size & Max: The number of people curently in the party and the maximum party size respectively

• Join Secret: This will be the Join Secret of the party. It can be left empty
                        Note: You can fill this in with a random string if you want to show the 'Ask to join' button

• Details: This will be the text that shows right under the app name

• State: This text will show up under Details
